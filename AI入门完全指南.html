🔍 概念百科

把所有提到的工具、模型、术语一次讲清楚

🛠️ 工具 & 平台

Trae IDE 工具
一款内置 AI 助力的编程软件（类似 VS Code 但自带 AI）。你可以直接在上面写代码、跟 AI 对话让它帮你写代码。支持绑定各类大模型。
操作要点：从官网下载，安装后右上角设置 → Editor → 字体调大（16），关闭 Auto 模式可看到内置免费模型。
🔗 关联：需要绑定 DeepSeek 等大模型才能发挥全部功能
CNB（cnb.cool）平台
腾讯旗下的代码托管平台，类似 GitHub 但国内访问更快更稳定。用来存放项目文件，支持版本管理。
操作要点：注册 → 创建仓库 → 生成"访问令牌"（Access Token）→ 让 AI 帮你克隆/上传。
🔗 关联：配合 Git 使用，AI 可通过指令 帮我 git 到 CNB 仓库 上传代码
网易云游戏 / 云电脑 工具
通过浏览器访问一台远程电脑桌面（云电脑），适合低配电脑用户。
操作要点：访问 CG.163.com → 云电脑 → 选普通配置（不要选"自带 AI"选项）→ 秒玩进入。
🔗 关联：低配电脑跑不动 AI 工具时的备选方案
MinerU 工具
一个能将 PDF 文件精准转换为 Markdown（.md）文档的 AI 工具。能识别章节结构、标题层级。
操作要点：官网注册 → 获取 API Key → 把 PDF + API 文档链接发给 AI，让它调用 MinerU 接口转换。
🔗 关联：PDF → Markdown → AI 读取更准确，是绘本制作的前提步骤
云雾 聚合平台
一个"模型聚合平台"——它代理了多家大模型（DeepSeek、阿里、豆包、GPT 等），你只需在云雾注册一个账号、充值一次，就能调用所有模型的 API。
操作要点：注册 → 添加令牌 → 复制 API Key + Base URL → 在 Trae 中配置。
🔗 关联：统一管理多个模型，切换时只需改"模型名称"
Live Server 插件
Trae IDE 中的一个扩展插件。安装后，右键 HTML 文件选择"Open with Live Server"，就能在浏览器中实时预览网页效果。
🔗 关联：预览 AI 生成的 HTML 交互页面时使用

🤖 大模型（AI 的大脑）

DeepSeek 大模型
国产大语言模型，擅长编程、文字对话、逻辑推理。课程中使用 DeepSeek-V3 / V3-Flash 版本。性能强且价格便宜（最低充值 10 元）。
关键：需要去 DeepSeek 官网申请 API Key，然后在 Trae 中绑定使用。
🔗 关系：课程中最核心的大模型，其他模型作为补充
MiniMax / GLM 大模型
Trae IDE 内置的免费模型，无需自己配置即可直接使用。高峰期可能需要排队等待。
🔗 关系：Trae 自带的备选模型，适合快速试用
Kimi / 豆包 / 智谱 / 小米 / 硅基流动 大模型
其他国产大模型，各有特色：Kimi 擅长长文本、豆包有作图能力、智谱是清华系等。在 Part 2 中被作为"自定义模型"的示例讲解。
🔗 关系：可互相替代，通过同一套方法（API Key + Base URL + 模型 ID）接入 Trae
GPT Image 2 大模型
OpenAI 的图像生成模型，课程中推荐的作图模型（绘本制作使用）。中等画质约 0.5~0.6 元/张。
注意：纯文本模型不能生成图片，必须接入专门的作图模型。
🔗 关系：在云雾等聚合平台中配置为"构图模型"使用

📚 专业术语

API 术语
全称 Application Programming Interface（应用程序接口）。简单理解：一个"通道"，让不同的软件之间可以互相沟通。你通过 API 告诉大模型"帮我写个程序"，大模型就把结果返回给你。
🔗 关联：使用任何大模型都需要通过 API
API Key（API 密钥 / 令牌 / Token） 术语
相当于你的"数字身份证 + 密码"。当你调用某个大模型的 API 时，需要出示 API Key 来证明你是谁以及你充了多少钱。
⚠️ 铁律：创建后只显示一次！必须立即复制保存到本地文件，关闭后就再也看不到了。
🔗 关联：配模型时需要填写的三要素之一
Base URL（接口地址） 术语
API 服务的"门牌号"——告诉软件去哪里找到这个模型。通常格式如 https://api.xxx.com/v1。
找不到时：去官网的"开发者文档"找，或直接把文档发给 AI 让它帮你提取。
🔗 关联：配模型时需要填写的三要素之二
模型 ID（模型名称） 术语
每个大模型都有一个官方唯一的名字（区分大小写），比如 deepseek-chat、gpt-image-2。配置时必须填写准确。
操作：去官网"模型列表/接口文档"复制官方名称。
🔗 关联：配模型时需要填写的三要素之三
Git 术语
一个版本管理工具，就像游戏里的"存档/读档"功能。每次修改完代码，用 Git 保存一个版本，以后可以随时回退。
操作：让 AI 帮你执行 Git 命令即可，无需手动掌握。
🔗 关联：搭配 CNB 仓库使用，实现云端备份和版本管理
Markdown（.md） 术语
一种轻量级文本格式，用简单的符号标记标题、列表等（如 # 标题、**加粗**）。AI 读取 .md 文件比 PDF 更准确。
操作：在 Trae 中右键 → 新建文件，以 .md 结尾命名。
🔗 关联：PDF → Markdown 是 AI 处理文档的标准预处理步骤
Auto 模式 / Solo 模式 术语
Trae IDE 中的两种工作模式。Auto 模式自动选择模型，建议关闭 Auto 手动选择。Solo 模式是日常编程推荐模式。
🔗 关联：日常使用建议 Solo 模式 + 手动选模型
Skill（快捷指令） 术语
把一套固定的操作流程打包成一个"快捷指令"。比如绘本制作流程验证后，让 AI 写成 Skill，以后直接调用就能自动执行。
🔗 关联：Part 6 中提到的自动化进阶操作
# 引用按钮 操作
在 Trae 对话框左下角的 # 按钮，点击可选择文件/文件夹附加到对话中。作用相当于把资料"递给"AI 看，比手动拖拽更方便，打开大文件不会卡死。
🔗 关联：替代双击打开大文件，低配电脑尤其有用

📋 小白操作路线图

从零开始的完整操作步骤

1

🖥️ 装好环境

  • 下载安装 Trae IDE（从官网）
  • 字体调大（16）、界面缩放（Zoom 1~2）
  • 关闭 Auto 模式，用 Solo 模式
  • 低配电脑先用云电脑（CG.163.com）
2

🔑 搞定 API Key

  • DeepSeek 官网 → API 开放平台 → 创建 API Key
  • ⚠️ 立即复制保存到本地（只显示一次！）
  • 在 Trae 中添加模型 → 选 DeepSeek → 粘贴 Key
  • 充 10 元余额即可使用
3

🧪 开始使用 AI

  • 在 Trae 对话区输入你的需求
  • 用 # 引用按钮添加文件到对话
  • 让 AI 读文件、写代码、生成页面
  • 不满意就直接说，让 AI 改到满意
4

📦 管理项目和素材

  • 注册 CNB → 创建仓库 → 生成访问令牌
  • 让 AI 克隆课程素材/上传代码
  • 每完成一版让 AI "git 到仓库"
  • 需要旧版本时让 AI 回退即可
5

📖 PDF 转文本（进阶）

  • 注册 MinerU → 获取 API Key
  • 把 PDF + API 文档发给 AI
  • 让 AI 调用 MinerU 转成 Markdown
  • 或用云雾聚合平台统一管理
6

🎨 绘本制作（实战）

  • 准备参考图 + 转好的 MD 文档
  • 先让 AI 写需求文档（省成本）
  • 配置作图模型（如 GPT Image 2）
  • 先做第一张检查 → 满意后批量生成
  • 最终合成 PDF 绘本

🎯 小白锦囊

  • 不用怕—— 你不需要懂编程，AI 会帮你写代码，你只需要会说话、能描述需求就行。
  • API Key 是命根子—— 创建后只显示一次，务必立即复制存到本地文件，关了就没了。
  • 先看结果再批量—— 让 AI 先做一小部分你看看效果，满意了再让它做全部，避免浪费时间和金钱。
  • 把 AI 当实习生—— 它做错了直接告诉它哪里不对，让它重新做。不满意就反复改。
  • 大文件不要双击打开—— 在 Trae 中用 # 引用文件到对话，防止卡死。
  • 说"中文"—— 提示词里记得加一句"我要中文的"，不然 AI 可能会给你输出英文。
  • 新任务新对话—— 不同项目就新建一个对话窗口，避免上下文互相干扰。
  • 遇到问题问 AI—— 安装失败？直接跟 AI 说"帮我把 Git 环境装好"，它能搞定大多数技术问题。
